# 🚖 OLA Ride Booking Data Analysis Dashboard

A complete **Data Analytics Project** built using **SQL, Excel, and Power BI** to analyze OLA ride booking data. This project provides business insights into booking performance, revenue, cancellations, customer behavior, driver performance, and vehicle utilization through interactive dashboards.

---

## 📌 Project Overview

The objective of this project is to analyze OLA booking data and create an interactive Power BI dashboard that helps stakeholders understand:

- Overall booking performance
- Revenue analysis
- Ride cancellation trends
- Customer and driver ratings
- Vehicle type performance
- Payment method analysis
- Distance travelled analysis

---

## 🛠️ Tech Stack

- **SQL (MySQL)** – Data querying and analysis
- **Power BI** – Interactive dashboard and visualization
- **Microsoft Excel** – Dataset
- **DAX** – Measures and calculations
